About Docker Project
This is a graduation project for DevOps course, sponsored by the Egyptian National Council and the United Nations.
At this project I created a Dockerfile to build an image based on "python:3.7-alpine" and start a new container based on this image
To build an image from the Dockerfile run this command

docker image build -t <imageName>:<imageTag> .

To start a new container based on the created image run this command

docker container run -d --name <containerName> -p 5000:5000 <imageName>:<imageTag> .
